# Agent Workflow Landing Page

A small static portfolio demo for the CO/AI Vibe Coder application path.

This is a self-made AI-assisted frontend workflow example. It is not client work, commissioned work, or a shipped production project.

## Concept

The page presents an "AI Agent Workflow Dashboard" for a junior AI agent operator. It shows a simple process for taking a task from intake through repo inspection, AI drafting, human review, verification, and final reporting.

## Built With

- HTML
- CSS
- JavaScript
- Codex
- ChatGPT
- GitHub
- Browser
- Local workspace workflow

No paid tools, external API keys, backend services, or complex frameworks are required.

## Features

- Responsive landing page
- Workflow step cards
- Category filters for workflow cards
- Review checklist with progress tracking
- Copyable workflow prompt
- Clear self-made demo positioning

## Human-Reviewed Items

- Page text avoids fake client claims.
- The demo is described as portfolio practice.
- Layout is responsive for desktop and mobile widths.
- JavaScript is limited to simple, reviewable interactions.
- No sensitive data, API keys, tracking, payment flows, or external service dependency are included.

## How to Run Locally

Open `index.html` in a browser.

No build step is needed.

## Suggested Review Steps

1. Open `index.html`.
2. Test the workflow card filters.
3. Toggle the review checklist and confirm the progress meter updates.
4. Click the copy prompt button.
5. Resize the browser or test mobile width.
6. Read `workflow_notes.md` to understand the AI-assisted process.

## Notes

This project is meant to show responsible AI-assisted workflow, not senior frontend engineering. It is intentionally small, readable, and easy to inspect.
