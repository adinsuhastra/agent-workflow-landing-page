How this demo works

AI drafts, human judgment decides.

The workflow keeps AI assistance bounded: Codex or ChatGPT can help plan, draft, and inspect changes, but the final deliverable is checked by a human before it is treated as ready.

Workflow steps

From task intake to delivery report

01

Task Intake

Clarify the requested outcome, constraints, approval rules, and any sensitive actions that require stopping first.

02

Repo Inspection

Review the current files, structure, patterns, and local instructions before deciding how to change anything.

03

AI Drafting

Use AI to create a small draft, plan, or code change in reviewable steps instead of large unchecked output.

04

Human Review

Inspect content, code, links, claims, layout, and risk before accepting the AI-assisted draft as usable.

05

Testing / Verification

Run available checks, open the page locally, test interactions, and look for visible layout or text issues.

06

Delivery Report

Summarize what changed, what was verified, known limits, and the next approval step before publishing.
